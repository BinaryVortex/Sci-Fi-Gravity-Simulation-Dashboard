# 🌌 Sci‑Fi Gravity Simulation Dashboard

A futuristic, interactive gravity simulation dashboard built with **HTML, CSS, and JavaScript**. This project combines real-time gravitational physics visualization with a sleek sci‑fi interface, allowing users to explore celestial motion, orbital mechanics, and dynamic particle interactions directly in the browser.

## 🚀 Overview

Sci‑Fi Gravity Simulation Dashboard is an immersive web application designed to demonstrate the principles of gravity through an engaging science-fiction themed user interface.

Whether you're a student, developer, physics enthusiast, or someone who enjoys interactive visualizations, this project offers a unique way to explore gravitational behavior and motion in a visually appealing environment.

## ✨ Features

- 🌍 Real-time gravity simulation
- 🛰️ Interactive celestial body visualization
- 📊 Futuristic dashboard interface
- ⚡ Smooth animations and responsive controls
- 🎨 Sci‑fi inspired design language
- 📱 Responsive layout for different screen sizes
- 🌌 Dynamic visual effects and immersive experience
- 🔧 Built entirely with vanilla HTML, CSS, and JavaScript

- **HTML5** – Structure and layout

- **JavaScript (ES6+)** – Simulation logic and interactivity

## ⚙️ How It Works

The simulation calculates gravitational interactions between objects and updates their positions continuously to create realistic motion effects.

Core concepts demonstrated:

- Universal Gravitation
- Orbital Motion
- Velocity and Acceleration
- Force-Based Movement
- Real-Time Rendering

The dashboard visualizes these calculations through animated graphical elements and futuristic UI components.
